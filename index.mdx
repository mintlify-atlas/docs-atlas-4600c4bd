  <Steps>
    <Step title="Clone the repository">
      Start by cloning the project from GitHub:
      
      ```bash
      git clone https://github.com/IscMiguelAngel/advice-generator.git
      cd advice-generator
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required packages using npm:
      
      ```bash
      npm install
      ```

      This will install React 18, React DOM, Vite, and SASS.
    </Step>

    <Step title="Start the development server">
      Run the Vite development server:
      
      ```bash
      npm run dev
      ```

      Your app will be available at `http://localhost:5173`
    </Step>

    <Step title="Try it out">
      Click the dice button to fetch a new piece of advice from the Advice Slip API. The button animates while loading!
      
      <Note>
        The app fetches advice from `https://api.adviceslip.com/advice` — no API key required.
      </Note>
    </Step>
  </Steps>
