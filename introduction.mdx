The Advice Generator is a modern React application that fetches and displays random advice using the Advice Slip API. Built as a Frontend Mentor challenge, it demonstrates clean component architecture, API integration, and responsive design principles.

## What is Advice Generator?

Advice Generator is a simple yet elegant web application that delivers random pieces of advice at the click of a button. Each time you click the dice button, the app fetches a new piece of advice from the Advice Slip API and displays it in a beautifully designed card interface.

## Tech stack

The project is built with modern web technologies:

- **React 18.2** - Component-based UI library
- **Vite** - Lightning-fast build tool and development server
- **SASS** - CSS preprocessor for maintainable styles
- **Advice Slip API** - Free API for random advice quotes

<Note>
  This project uses React 18's new `createRoot` API for improved rendering performance.
</Note>

## Project structure

The application follows a clean, organized structure:

```
src/
├── components/
│   └── card.jsx          # Main card component
├── stylesheets/
│   └── styles.scss       # SASS styles
├── assets/               # SVG icons and images
└── main.jsx              # Application entry point
```

## Design specifications

The project includes a comprehensive style guide:

- **Typography**: Manrope font family with 800 weight
- **Primary colors**: Light Cyan `hsl(193, 38%, 86%)` and Neon Green `hsl(150, 100%, 66%)`
- **Neutral colors**: Dark Blue `hsl(218, 23%, 16%)` for backgrounds
- **Font size**: 28px for quote text

<Warning>
  The Advice Slip API has rate limiting. Be mindful of request frequency during development.
</Warning>
