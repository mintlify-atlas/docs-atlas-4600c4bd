This guide will help you set up and run the Advice Generator app on your local machine. You'll have a fully functional development environment in under 5 minutes.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v14 or higher)
- **npm** (comes with Node.js)
- **Git** (for cloning the repository)

<Note>
  You can verify your Node.js installation by running `node --version` in your terminal.
</Note>

## Installation

<Steps>
  <Step title="Clone the repository">
    Clone the Advice Generator repository to your local machine:

    ```bash
    git clone https://github.com/IscMiguelAngel/advice-generator.git
    cd advice-generator
    ```
  </Step>

  <Step title="Install dependencies">
    Install all required npm packages:

    ```bash
    npm install
    ```

    This will install:
    - React 18.2 and React DOM
    - Vite for development and building
    - SASS for CSS preprocessing
    - Required Vite plugins
  </Step>

  <Step title="Start the development server">
    Launch the Vite development server:

    ```bash
    npm run dev
    ```

    You should see output similar to:

    ```
    VITE v3.1.0  ready in 500 ms

    ➜  Local:   http://localhost:5173/
    ➜  Network: use --host to expose
    ```
  </Step>

  <Step title="Open in your browser">
    Navigate to `http://localhost:5173/` in your web browser. You should see the Advice Generator app with a card displaying random advice.

    Click the dice button to fetch new advice!
  </Step>
</Steps>

```
├── src/

│   ├── components/
│   │   └── card.jsx      # Main Card component
│   ├── stylesheets/
│   │   └── styles.scss   # SASS styles
│   └── main.jsx          # App entry point
├── index.html            # HTML template
├── package.json          # Dependencies and scripts
└── vite.config.js        # Vite configuration
```

## Understanding the entry point

The application starts in `src/main.jsx`:

```jsx src/main.jsx
import React from 'react'
import ReactDOM from 'react-dom/client'
import Card from './components/card'
import "./stylesheets/styles.scss"

ReactDOM.createRoot(document.getElementById('root')).render(
  <React.StrictMode>
    <Card />
  </React.StrictMode>
)
```

This file:
- Uses React 18's `createRoot` API for improved performance
- Renders the `Card` component inside React.StrictMode
- Imports global SASS styles

## Available scripts

The `package.json` defines three npm scripts:

<CodeGroup>
```bash Development
npm run dev
```

```bash Production build
npm run build
```

```bash Preview build
npm run preview
```
</CodeGroup>

### Script descriptions

<AccordionGroup>
  <Accordion title="npm run dev">
    Starts the Vite development server with hot module replacement (HMR). Changes to your code will automatically refresh in the browser.
  </Accordion>

  <Accordion title="npm run build">
    Creates an optimized production build in the `dist/` directory. This minifies JavaScript, optimizes assets, and prepares the app for deployment.
  </Accordion>

  <Accordion title="npm run preview">
    Serves the production build locally for testing before deployment. Run `npm run build` first.
  </Accordion>
</AccordionGroup>

## Testing the API

The app fetches advice from the Advice Slip API. You can test the API directly:

```bash
curl https://api.adviceslip.com/advice
```

Expected response:

```json
{
  "slip": {
    "id": 117,
    "advice": "It is easy to sit up and take notice, what's difficult is getting up and taking action."
  }
}
```

<Warning>
  The Advice Slip API may cache responses briefly. If you're not seeing new advice immediately, this is normal behavior.
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port 5173 is already in use">
    Vite will automatically try the next available port (5174, 5175, etc.). Check the terminal output for the correct URL.
  </Accordion>

  <Accordion title="SASS compilation errors">
    Ensure SASS is properly installed:
    ```bash
    npm install sass
    ```
  </Accordion>

  <Accordion title="API not responding">
    Check your internet connection. The Advice Slip API requires an active connection. The app will show loading states if the API is slow.
  </Accordion>
</AccordionGroup>
